# NÃO ENTRE EM PÂNICO!
## Um jogo da memória com uma interface retrofuturista inspirada em dispositivos de sci-fi clássico.

-------------------------------------------

### Sobre o projeto
Este projeto é um jogo da memória com foco em experiência visual e interatividade, explorando uma estética retrofuturista inspirada em dispositivos sci-fi clássicos.
Teve como ponto de partida um desafio proposto pela plataforma Digital Innovation One (DIO). A partir da base original, foram realizadas diversas melhorias e modificações, incluindo:
  - Reformulação completa da interface
  - Adição de sistema de tempo (timer)
  - Implementação de feedback visual com LEDs
  - Criação de identidade visual própria
  - Ajustes de responsividade e experiência do usuário

O objetivo foi ir além da proposta inicial, explorando personalização, design e interatividade.

A proposta foi evoluir um projeto base simples para algo com identidade própria, incorporando elementos visuais, feedback em tempo real e pequenos detalhes que enriquecem a experiência do usuário.


As referências implícitas nesse projeto podem parecer... estranhamente específicas. 😉

-------------------------------------------

### Funcionalidades
- Sistema de cartas com lógica de comparação
- Timer com contagem regressiva
- Feedback visual com LED:
  - Acerto
  - Erro
- Alerta nos últimos segundos
- Interface estilizada com animações

-------------------------------------------

### Tecnologias utilizadas
- HTML5
- CSS3 (Flexbox, animações, responsividade)
- JavaScript (manipulação de DOM e lógica do jogo)

-------------------------------------------

### Responsividade
O jogo foi adaptado para diferentes tamanhos de tela, com:
- Tipografia fluida usando clamp()
- Ajustes no visor para evitar distorções no mobile
- Layout flexível com melhor aproveitamento de espaço

-------------------------------------------

### Aprendizados
Durante o desenvolvimento, foram explorados:
- Manipulação de DOM
- Controle de estado
- Lógica de jogo
- Design de interface (UI/UX)
- Responsividade
